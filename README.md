# [1주차] 시간 복잡도 & 공간 복잡도
- 시간 복잡도 : 입력값(N)에 비례해서 어느 정도로 증가하는지 파악
- 공간 복잡도 : 대부분의 문제에서는 알고리즘의 성능이 공간에 의해 결정x, but N^2, N^3 일 경우는 달라질수 있다.
- 점근 표기법
-   빅오(O(N)) 표기법 : 최악의 성능이 나올때 어느 정도의 연산량이 나올지 표기
-   빅오메가(Ω(1)) 표기법 : 최선의 성능이 나올때 어느 정도의 연산량이 나올지 표기
